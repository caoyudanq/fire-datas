# 火警大数据平台

## 项目介绍

本项目基于 vue 框架，完成某市符合国家消防规定单位的火警信息，以及火警信息的历史记录、实时报警记录展示功能。

## 项目构建

```bash
npm install
npm run dev
```

PS: 默认 8080 端口

## 项目技术栈

- vue-cli
- vue
- vue-axios
- vue-router
- webpack
- Vuex
- Element UI
- Echarts
- express
